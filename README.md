### GSCMinify

GSC minifier and mangler built for use with Infinity Loader projects.

## NOT FOR PRODUCTION. IN DEVELOPMENT MODE.

## Run Locally

```sh
npm i
```

```sh
node index.js --input <filename.gsc>
```


## Prepare for release

```sh
npm i pkg -g
```
```sh
pkg index.js 
```
